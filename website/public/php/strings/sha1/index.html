How to use

npm is the recommended installation method. It pairs nicely with a CommonJS module bundler such as Browserify, webpack or rollup.js.

You you can install via npm install --save --save-exact locutus and require it like so: var sha1 = require('locutus/php/strings/sha1') and start using sha1. You could also require the strings module in full.
